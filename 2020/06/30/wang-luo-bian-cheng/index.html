网络编程

概述

计算机网络

计算机网络是指将地理位置不同的具有独立功能的多台计算机及其外部设备，通过通信线路连接起来，在网络操作系统，网络管理软件及网络通信协议的管理和协调下，实现资源共享和信息传递的计算机系统。

网络编程的目的

无线电台，传播交流信息。数据交换，通信

想要达到这个效果需要什么

  1. 如何准确地定位网络上的一台主机，唯一的IP地址，端口，定位到这个计算机上的某个资源
  2. 找到了这个主机，如何传输数据

IP

IP地址： InetAddress

  • 唯一定位一台计算机

  • 127.0.0.1 : 代表本机

  • IP地址的分类

    1. IP地址分类 ipv4 / ipv6

      ipv4：127.0.0.1，有四个字节组成，（1字节的范围是0-255（8位，2^8），大概有42亿个,255的四次方。对于地球来说不够用）在2011年就用尽了

      ipv6：dos命令：ipconfig，查看本机ip.128位，八个无符号十六进制整数

      ipv6的写法：

      例如这样一个ipv6地址
      2001 : 0db8 : 85a3 : 0000 : 1319 : 8a2e : 0370 : 7344
      一共八组，每组4个数字，一共32个数，每个数字都是十六进制数，一个十六进制数可以写成4个二进版制数（就是十六进制数转成二进制数）
      可以理解为有多少位就是多少个二进制数
      所以，32×4=128位

    1. 公网（互联网）/ 私网（局域网）分类
      • 192.168.xx.xx开头一般都是局域网，专门给组织内部使用
      • ABCD IP地址分类

常用的一个类：InetAddress

import java.net.InetAddress;
import java.net.UnknownHostException;

//测试IP
public class TestInetAddress {
    public static void main(String[] args) {
        try {
            //查询本机地址
            InetAddress inetAddress1 = InetAddress.getByName("127.0.0.1");
            //InetAddress这个类没有构造器，可以通过类名的getByName或地址方法返回它的对象
            System.out.println(inetAddress1);

            InetAddress inetAddress3 = InetAddress.getLocalHost();
            System.out.println(inetAddress3);

            InetAddress inetAddress4 = InetAddress.getByName("localhost");
            System.out.println(inetAddress4);

            //查询百度的ip地址
            InetAddress inetAddress2 = InetAddress.getByName("www.baidu.com");
            System.out.println(inetAddress2);

            //常用方法
            //System.out.println(inetAddress2.getAddress());   不用记
            System.out.println(inetAddress2.getCanonicalHostName()); //获得规范的名字（ip）
            System.out.println(inetAddress2.getHostAddress());       //获得ip
            System.out.println(inetAddress2.getHostName());          //获得机主名/域名（网址）
        } catch (UnknownHostException e) { //未知的主机异常
            e.printStackTrace();
        }

    }
}

1.4端口

端口表示计算机上的一个程序的进程；

  • 不同的进程有不同的端口号，所以端口号是不能冲突的。端口用来区分软件。

  • 端口号被规定为0-65535

  • 端口分为TCP端口和UDP端口，每一种都有65535上限，单个协议下面，端口号不能冲突（同一台计算机上不能有两个相同的端口号）

  • 端口分类

    • 公有端口 0-1023
    • HTTP：80
    • HTTPS：43（更安全的协议）
    • FTP:21
    • TELENT(远程监听窗口)：23

  • 程序注册端口：1024-49151（这些端口一般是分配给用户或者程序的）

    • Tomcat：端口号8080
    • MySQL：3306
    • Oracle：1521

    端口号不用强行记忆，用多了就会了

    • 动态、私有端口：49152-65535
    • 常用的DOS命令
    netstat -ano ：#查看所有的端口
netstat -ano findstr "5900" #查看所有5900端口的进程
tasklist findstr "8696" #查看指定端口的进程

    一般端口和ip地址是连起来写的，格式为 ip地址：端口

    比如：192.168.1.1 : 65536

    CTRL + shift + ESC = 打开任务管理器

1.5通信协议

协议：约定，就好比我们现在说的是普通话

网络通信协议：针对网络产生的协议，比如，速率，传输码率，代码结构，传输控制……

问题非常的复杂，是通信工程专业学的

大事化小：分层

TCP / IP 协议簇：实际上是一组协议

  • TCP：用户传输协议 （要接电话）
  • UDP：用户数据报协议 （给你发短信）

出名的协议：

  • TCP
  • IP：网络互连协议

TCP和UDP的对比

TCP：打电话

  • 连接，稳定

  • 三次握手，四次挥手

    • 连接：
A：你瞅啥？
B：瞅你咋地？
A：干一场！


断开：
A:我要断开了
B：我知道了
B:你已经断开了吗
A：已经断开了

最少需要三次才能建立连接

  • 客户端，服务端

  • 传输完成，会释放连接，效率低

UDP:发短信

  • 不连接，不稳定
  • 客户端，服务端：没有明确的界限
  • 不管有没有准备好，都可以发给你
  • 导弹
  • DDOS：洪水攻击，会造成端口堵塞。饱和攻击

1.6TCP

TCP需要一边发送一遍接受，所以需要一个客户端，一个服务器

客户端

  1. 通过Socket连接服务器
  2. 发送消息

服务器

  1. 建立服务端口ServerSocket
  2. 等待用户连接。通过accept监听
  3. 接受用户的消息

客户端：

import java.io.IOException;
import java.io.OutputStream;
import java.net.InetAddress;
import java.net.Socket;
import java.net.UnknownHostException;

//客户端
public class TCPClientDemo01 {
    public static void main(String[] args) {
        Socket socket = null;
        OutputStream os = null;
        //1.客户端要连接服务器，要知道服务器的地址
        try {
            //这个是获取本机的地址
            InetAddress serverIP = InetAddress.getByName("127.0.0.1");
            //2.端口号
            int port = 54532;
            //创建一个Socket连接，客户端的创建连接类为Socket（一个插头）
            //可以默认不传参，也有可以传IP和端口的方法，先IP后端口，方便建立链接
            socket = new Socket(serverIP,port);
            //3.连接建立完毕，发送消息 （IO流)
            //socket要向外写东西，所以要用OutputStream
            os = socket.getOutputStream();
            os.write("你好！".getBytes()); //getBytes()让字符串变成字符数组

        } catch (UnknownHostException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace(); //这里alt+Enter自动出来了这个，其实可以直接把上面那个catch里的异常
                                 //提升一下作用域，换成IOException就可以捕获到了
                                 //一个try可以有多个catch
        } finally{
            //客户端也要关闭资源
            if(os != null) {
                try {
                    os.close();
                } catch (IOException e) {
                    e.printStackTrace();
                } if (socket != null) {
                    try {
                        socket.close();
                    } catch (IOException e) {
                        e.printStackTrace();
                    }
                }
            }
        }
    }
}

服务端：

import java.io.ByteArrayOutputStream;
import java.io.IOException;
import java.io.InputStream;
import java.net.ServerSocket;
import java.net.Socket;

//服务端
public class TCPServerDemo01 {
    public static void main(String[] args) {
        ServerSocket serverSocket = null;
        Socket socket = null;
        InputStream is = null;
        ByteArrayOutputStream baos = null;
        //1.服务器对应地来说要有一个地址，创建一个地址的对应方法为ServerSocket()
        //Socket的意思是套接字（插头），整个Java的网络连接就像一个个的插头一样
        //里面传的这个参数是端口号
        //要捕获一个异常
        try {
            serverSocket = new ServerSocket(54532);
            //2.等待客户端连接过来，客户端有一个接收方法accept()返回一个Socket对象，此时这个Socket就是客户端的Socket
            //在这上面加一个while(true,则可以不停地接收消息（一直accept监听）
            socket = serverSocket.accept();
            //3.读取客户端的消息
            is = socket.getInputStream();//这两个是同一个，那边传过来，这边读进来
            //正常情况下请用管道流读取数据，不要用字符数组那个1024，那个low了，而且容易出错
            //管道流
            baos = new ByteArrayOutputStream();
            byte[] buffer = new byte[1024];     //缓冲区
            int len;
            while ((len = is.read(buffer)) != -1) {
                baos.write(buffer,0,len);

                //先开的后关
                //先开后关的意思是有可能管道里面还有水，你得一节一节地关掉
                //这四个东西都要放到finally里面，但是它们在try的代码块里面，放不进去，所以要把定义放在上面，提升作用域
            }
            System.out.println(baos.toString());
        } catch (IOException e) {
            e.printStackTrace();
        } finally { //所有东西写完了需要关闭流，所以这些操作放到finally里面
            //关闭资源
            if(baos != null) {
                try {
                    baos.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
                if(is != null);{
                try {
                    is.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
            if(socket != null){
                try {
                    socket.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            } if (serverSocket != null) {
                try {
                    serverSocket.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }
    }
}

服务器端：

import java.io.FileOutputStream;
import java.io.IOException;
import java.io.InputStream;
import java.io.OutputStream;
import java.net.ServerSocket;
import java.net.Socket;

public class TCPServerDemo02 {
    public static void main(String[] args) throws Exception {
        //创建服务
        ServerSocket serverSocket = new ServerSocket(10094);
        //监听客户端的连接
        Socket socket = serverSocket.accept(); //阻塞式监听，会一直等待客户端连接，连接完监听结束
        //3.获取输入流
        InputStream is = socket.getInputStream();
        //4.文件输出
        FileOutputStream fos = new FileOutputStream("receive.jpg");
        byte[] buffer1 = new byte[2048];
        int len;
        while ((len = is.read(buffer1)) != -1) {
            fos.write(buffer1, 0, len);
        }

        //通知客户端我接收完毕了,通知的话需要进行输出
        OutputStream os = socket.getOutputStream();
        os.write("我接收完毕了，你可以断开了".getBytes());


        //关闭资源
        fos.close();
        is.close();
        socket.close();
        serverSocket.close();
    }
}

import java.io.*;

public class TCPClientDemo02 {
    public static void main(String[] args) throws Exception {
        //1.创建一个Socket连接
        Socket socket = new Socket(InetAddress.getByName("127.0.0.1"), 10094);
        //2.创建一个输出流，写东西
        OutputStream os = socket.getOutputStream();

        //文件首先要读取，先把文件变成一个流再把这个流输出去
        FileInputStream fis = new FileInputStream(new File("C:\\Users\\xiaoqiyao\\OneDrive\\桌面\\算法.jpg"));
        //写出文件（写出文件用输出流来写）
        byte[] buffer = new byte[2048];
        int len;
        while ((len = fis.read(buffer)) != -1) {
            os.write(buffer, 0, len);
        }

        //通知服务器，我已经传输完了
        socket.shutdownOutput(); //我已经传输完了


        //确定服务端接收完毕才能断开连接

        InputStream is = socket.getInputStream();
        //String byte[]
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        byte[] buffer02 = new byte[1024];
        int len02;
        while ((len02 = is.read(buffer02)) != -1) {
            baos.write(buffer02,0,len02);
        }

        System.out.println(baos.toString());

        //5.关闭资源
        baos.close();
        is.close();
        fis.close();
        os.close();
        socket.close();
    }
}

发短信：不用连接，但是需要知道对方的地址

发送消息

import java.net.*;

//不需要连接服务器
public class UDPClientDemo01 {
  public static void main(String[] args) throws Exception {
      //1.建立一个Socket套接字
      DatagramSocket datagramSocket = new DatagramSocket();

      //2.建个包
      String message = "你好啊，服务器";

      //发送给谁（需要创建一个服务器地址）
      InetAddress localhost = InetAddress.getByName("127.0.0.1");
      int port = 9090;

      //参数： 数据，数据的长度，要发送给谁
      DatagramPacket datagramPacket = new DatagramPacket(message.getBytes(),0,message.getBytes().length,localhost,port);

      //3.发送包
      datagramSocket.send(datagramPacket);

      //4.关闭资源
      datagramSocket.close();
  }
}

接收端

import java.net.DatagramPacket;
import java.net.DatagramSocket;

public class UDPServerDemo01 {
    public static void main(String[] args) throws Exception{
        //开放端口
        DatagramSocket datagramSocket = new DatagramSocket(9090);
        //接收数据包
        byte[] buffer = new byte[1024];
        DatagramPacket datagramPacket = new DatagramPacket(buffer,0,buffer.length );
        //接收数据包
        datagramSocket.receive(datagramPacket); //阻塞接收 （一直保持接受状态，只要你活着就能收到包裹）

        System.out.println(datagramPacket.getAddress());
        System.out.println(new String(datagramPacket.getData()));
        System.out.println(datagramPacket.getData().toString());

        //关闭连接
        datagramSocket.close();
    }
}

循环发送消息

发送方：

import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.net.DatagramPacket;
import java.net.DatagramSocket;
import java.net.InetSocketAddress;

public class UDPSenderDemo01 {
    public static void main(String[] args) throws Exception {
        DatagramSocket datagramSocket = new DatagramSocket(8888);

        //准备数据：控制台读取System.in

        BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));

        //循环发送
        while (true) {
            String data = reader.readLine();
            byte[] dataBytes = data.getBytes();

            DatagramPacket datagramPacket = new DatagramPacket(dataBytes, 0, dataBytes.length, new InetSocketAddress("127.0.0.1", 6666));

            datagramSocket.send(datagramPacket);

            if (datagramPacket.equals("bye")) {
                break;
            }
        }
        datagramSocket.close();
    }
}

接收方：

import java.net.DatagramPacket;
import java.net.DatagramSocket;

public class UDPReceiverDemo01 {
    public static void main(String[] args) throws Exception{
        DatagramSocket datagramSocket = new DatagramSocket(6666);


        while(true){

            //这个接收到的datagramPacket需要一个地方去放它
            //准备接收包裹
            byte[] container = new byte[1024];  //接收包裹的是一个容器

            DatagramPacket datagramPacket = new DatagramPacket(container,0, container.length);

            datagramSocket.receive(datagramPacket);
            //先receive在读取数据输出，接都没接收到怎么输出

            //怎么读呢？跟前面一样直接输出就行
            //但是我们要先做一个断开连接的判断,如果接收到bye则断开连接
            byte[] data = datagramPacket.getData();
            String receiveData = new String(data, 0, data.length);
            System.out.println(receiveData);

            if(receiveData .equals("bye")) {
                break;
            }

        }   //阻塞式接收包裹

        datagramSocket.close();

    }
}
